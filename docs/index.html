Security Publications & CVEs

CVE-2024-41226 HIGH SEVERITY

CSV Injection in Enterprise Automation Solution

Published: January 2024 Affected: Enterprise Automation Solution Type: CSV Injection / Code Execution

Discovered and responsibly disclosed a critical CSV injection vulnerability in Enterprise Automation Solution. The vulnerability allows attackers to execute arbitrary code via crafted CSV Injection payload in the user management export functionality. When exploited, this vulnerability enables arbitrary command execution at the client OS level after manipulating the exported file content.

Technical Details:

  • Affected Component: /usermanagement/users/export
  • Attack Vector: Crafted CSV file manipulation
  • Impact: Client-side code execution
  • Required Access: Administrative privileges
